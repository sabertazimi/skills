# Skills

Personal agent plugin marketplace.

## Installation

```bash
npx skills add sabertazimi/skills
```

### Claude Code

Register the marketplace:

```bash
claude plugin marketplace add sabertazimi/skills
```

Install the plugin:

```bash
claude plugin install sabertaz@sabertaz
```

Update the plugin:

```bash
claude plugin update sabertaz@sabertaz
```
